# Git Manager 开发工作流

## 启动开发服务器

```bash
cd D:\workspace\workspace_ai\demo_OpenSpec\git_tools\git-manager
wails dev
```

## 运行测试

```bash
# 后端测试
go test ./...

# 前端测试
cd frontend
npm test

# 查看测试覆盖率
npm run test:coverage
```

## 构建生产版本

```bash
wails build -clean
# 输出：build/bin/git-manager.exe (13MB)
```

## 运行应用

```bash
# 开发模式
wails dev

# 生产模式
build\bin\git-manager.exe
```
