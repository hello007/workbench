# WorkBench 功能说明

## 1. 工作目录管理

- 添加/删除/更新工作目录
- 设置默认目录
- 拖拽排序目录顺序
- JSON持久化配置
- 路径验证（Windows格式）
- 右键菜单：重命名、设为默认、在资源管理器中打开、用 VSCode 打开、用 Warp 打开、用 Obsidian 打开（以该目录作为 Obsidian 仓库）、更新仓库、删除

## 2. 文件树浏览

- 懒加载机制（按需加载子节点）
- Git仓库自动检测（✓标识）
- .git文件夹过滤
- 全部展开/收起功能
- 节点选择和高亮
- 点击祖先目录时仅选中不收起（向上导航保护）

## 3. 文件操作

- 创建文件/文件夹
- 重命名、删除
- 文件预览（支持文本/二进制检测）
- 文件预览编辑 — 预览文本文件时可就地编辑，修改后保存或取消，切换文件时自动检查未保存修改
- 文件大小限制（1MB）
- 剪切/复制/粘贴操作
- 拷贝到对话框（支持预览效果）
- 用默认程序打开文件
- 右键菜单操作（新建文件/文件夹、重命名、删除、剪切/复制/粘贴、拷贝到、复制路径/文件名、在资源管理器中打开、用 VSCode/Warp 打开、用 Obsidian 打开、更新仓库）
- 用 Obsidian 打开：文件夹→以自身作为仓库，文件→以父目录作为仓库
- 按节点类型区分菜单项（文件夹/文件显示不同操作）

## 4. Git集成

- 查看仓库信息（分支、远程、提交历史）
- 克隆远程仓库（HTTPS/SSH）
- 拉取更新
- 切换分支
- 查看本地变更列表
- 丢弃本地更改
- 分页查看提交历史（默认10条/页）
- 批量更新仓库（递归扫描子目录，并行拉取，实时进度展示）

## 5. 智能导航中心

- Command Palette 快速导航（Ctrl+P）
- 文件/目录模糊搜索
- 内容搜索（基于 ripgrep，支持文件扩展名过滤、排除规则配置、指定目录搜索）
- 收藏夹系统（添加、分组、跳转、右键移除）
- 工作目录快速切换（#前缀）
- 每个工作目录独立记忆展开状态
- 最近访问历史
- 深层子目录可作为工作目录
- 长路径省略显示（鼠标悬停显示完整路径）

## 6. 内置终端

- 底部面板集成完整终端，支持 PowerShell / CMD / Git Bash / WSL 切换
- 工作目录自动跟随文件树
- 可拖拽调节面板高度
- Ctrl+` 快速切换（快捷键可自定义）
- 首次打开即显示 Shell 提示符
- 设置面板可配置默认 Shell 类型和自定义路径

## 7. 自定义快捷键

- 设置面板「快捷键」标签页管理
- 支持「打开命令面板」和「切换终端」两个快捷键
- 点击录制新快捷键，实时显示组合键
- 右键菜单显示对应快捷键提示
- 单个重置 / 全部重置为默认值
- 配置持久化到 settings.json

## 8. 设置面板

- 弹窗式设置面板，左右双栏布局
- 通用设置、终端设置、搜索设置、快捷键设置四个标签页
- 通用→外部应用：可配置 Obsidian 程序路径，配置后"用 Obsidian 打开"优先使用该可执行文件；留空则尝试系统已注册的 Obsidian
- 设置项自动保存持久化
