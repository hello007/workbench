# Git Manager 功能说明

## 1. 工作目录管理

- 添加/删除/更新工作目录
- 设置默认目录
- JSON持久化配置
- 路径验证（Windows格式）

## 2. 文件树浏览

- 懒加载机制（按需加载子节点）
- Git仓库自动检测（✓标识）
- .git文件夹过滤
- 全部展开/收起功能
- 节点选择和高亮

## 3. 文件操作

- 创建文件/文件夹
- 重命名、删除
- 文件预览（支持文本/二进制检测）
- 文件大小限制（1MB）
- 右键菜单操作（新建文件/文件夹、重命名、删除、复制路径/文件名、在资源管理器中打开、更新仓库）
- 按节点类型区分菜单项（文件夹/文件显示不同操作）

## 4. Git集成

- 查看仓库信息（分支、远程、提交历史）
- 克隆远程仓库（HTTPS/SSH）
- 拉取更新
- 分页查看提交历史（默认10条/页）
- 批量更新仓库（递归扫描子目录，并行拉取，实时进度展示）
