# Git Manager 测试策略

## 单元测试

- **后端：** 测试模型验证、服务逻辑（15个测试用例）
- **前端：** 测试组件行为、用户交互、边界情况（41个测试用例）

## 集成测试

- **前后端绑定：** 测试Wails方法调用
- **文件操作：** 测试真实文件系统操作
- **Git集成：** 测试Git命令执行

## 测试覆盖

- 根节点检测逻辑
- 节点切换时预览清理
- 错误处理运算符优先级
- Debug工具功能
- 后端文件操作（创建/重命名/删除/OpenInExplorer）
- 后端Git扫描与批量拉取（ScanGitRepos/BatchPull）
- 前端右键菜单分发、重命名、删除、复制、资源管理器打开
